collect_metrics(metrics)¶

Like it’s name suggests the metrics requested are collected and returned. Must be overriden.

This method is called by the snapd deamon during the collection phase of the execution of a Snap workflow.

get_metric_types(config)¶

This method returns the metrics provided by this plugin. Must be overriden.

This method is called by the snapd deamon when the plugin is loaded. Requiring config to successfully return metric types should be avoided. Only in rare circumstances should a plugin require configuration data to load. See plugin authoring docs.

Parameters:config (snap_plugin.v1.ConfigMap) – Provides configuration data.
Returns:List of collected metrics.
Return type:list of snap_plugin.v1.Metric
