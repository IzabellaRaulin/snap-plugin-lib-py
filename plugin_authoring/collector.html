Collector¶

Below you will find details that cover writing a Snap collector plugin in Python.

Plugin Naming¶

The first naming decision you will need to make will be for your project/repository. For this we suggest the following format.

snap-plugin-<TYPE>-<PLUGIN_NAME>

TYPE is either collector, processor or publisher and the PLUGIN_NAME is the same name that your plugin will provide Snapd when it does its initial handshake (more on the later).

Creating a Class that inherits from snap_plugin.v1.Collector¶

In order to write a collector plugin you will want to start by defining a class that inherits from snap_plugin.v1.collector.Collector.
